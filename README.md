# ComfyUI-WP-nodes

ComfyUI自定义节点集合，为ComfyUI提供扩展功能。

## 节点列表

### 🍰Qwen-镜头预设
提供AI绘画中的镜头预设参数控制，包括：
- 镜头模式选择
- 专业镜头类型
- 移动控制（上移、下移、左移、右移）
- 角度控制（俯视、仰视）

## 安装方法
1. 将本仓库克隆到ComfyUI的custom_nodes目录
2. 重启ComfyUI

## 使用说明
1. 在ComfyUI界面中，从左侧节点菜单的`🍱ComfyUI-WP-nodes`分类中找到相关节点
2. 将节点拖入工作区，连接到文本处理或提示词输入
3. 根据需要调整参数
