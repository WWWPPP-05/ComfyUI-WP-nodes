# ComfyUI-WP-nodes

一个为ComfyUI设计的自定义节点集合，提供多种AI绘画辅助功能，包括镜头预设和打光预设。

## 项目特点

- 提供直观的UI控制界面
- 支持多种预设参数组合
- 生成高质量的提示词
- 分类清晰，易于使用
- 兼容ComfyUI-aki-v1.6版本

## 节点列表

### 🍰Qwen-镜头预设
提供AI绘画中的镜头预设参数控制，帮助用户快速调整相机视角和镜头效果。

**功能特性：**
- 镜头模式选择（标准、广角、长焦、鱼眼、微距、移轴等）
- 移动控制（上移、下移、左移、右移）
- 旋转控制（向左旋转、向右旋转）
- 视角控制（俯视图、仰视图、背面拍摄）
- 镜头拉近效果（支持特写镜头）

**参数说明：**
- 移动控制：开关，控制是否启用移动参数
- 向上移动：开关，启用俯视图效果
- 向下移动：开关，启用仰视图效果
- 向左旋转：滑块（0-90度），控制向左旋转角度
- 向右旋转：滑块（0-90度），控制向右旋转角度
- 背面：开关，启用背面拍摄效果
- 拉近镜头：滑块（0-100），控制镜头拉近程度，100时为特写镜头
- 镜头模式：下拉菜单，选择不同镜头类型

### 🍥Qwen-打光预设
提供多种预设的打光风格选项，帮助用户快速调整画面的光照效果。

**风格类别：**
- 柔和清新/治愈风格
- 戏剧张力/暗黑风格
- 经典肖像/专业棚拍风
- 梦幻奇幻/氛围风格
- 风格化/潮流街头风格

**参数说明：**
- 每个风格类别下有多种具体打光类型可供选择
- 支持多选不同风格的打光效果
- 自动生成详细的打光提示词

### 🍡Qwen-尺寸预设
提供多种预设尺寸选项，用于生成不同比例和分辨率的latent图像。

**功能特性：**
- 支持多种常用比例（1:1、2:3、3:2、3:4、4:3、9:16、16:9等）
- 多种分辨率可选
- 可设置批量张数
- 支持对齐到8的倍数（优化SD模型性能）

**参数说明：**
- 尺寸预设：下拉菜单，选择预设的尺寸和比例
- 批量张数：生成的图像数量（1-100）
- 对齐到8的倍数：开关，是否将尺寸调整为8的倍数

**输出：**
- latent：根据所选尺寸生成的潜在空间图像

## 安装方法

### 方法一：直接克隆
1. 打开终端，进入ComfyUI的`custom_nodes`目录
2. 执行克隆命令：
   ```bash
   git clone https://github.com/your-username/ComfyUI-WP-nodes.git
   ```
3. 重启ComfyUI

### 方法二：手动安装
1. 下载本仓库的ZIP文件
2. 解压到ComfyUI的`custom_nodes`目录下
3. 确保目录名称为`ComfyUI-WP-nodes`
4. 重启ComfyUI

## 使用说明

1. 在ComfyUI界面中，从左侧节点菜单的`🍱ComfyUI-WP-nodes`分类中找到相关节点
2. 将节点拖入工作区
3. 连接到文本处理或提示词输入节点
4. 根据需要调整参数
5. 运行工作流生成图像

## 示例工作流

1. **基础镜头控制**：使用「🍰Qwen-镜头预设」节点调整相机视角，连接到提示词输入
2. **专业打光设置**：使用「🍥Qwen-打光预设」节点选择合适的光照风格
3. **组合使用**：同时使用镜头和打光预设，创建更丰富的画面效果

## 系统要求

- Python 3.8+环境
